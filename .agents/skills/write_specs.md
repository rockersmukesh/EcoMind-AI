# Specification Writer

Before implementing any feature:

Create:

- Objective
- User Story
- Acceptance Criteria
- Edge Cases
- API Requirements
- UI Requirements
- Test Cases

Example:

Feature:
Carbon Twin Simulator

User Story:
As a user I want to simulate lifestyle changes
so I can understand future carbon impact.

Acceptance Criteria:

✓ User can modify travel habits

✓ AI predicts emissions

✓ Dashboard updates

✓ Results save to history
